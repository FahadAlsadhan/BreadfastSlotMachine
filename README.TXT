The Breadfast Slot Machine is a JavaFX-based interactive game created for
a marketing activation. It simulates a slot machine with custom graphics,
animations, and sound effects, designed to engage users and promote brand
visibility. Built with clean UI/UX, it integrates branding elements
seamlessly to deliver a fun, memorable experience.
